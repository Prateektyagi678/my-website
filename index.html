OpenFrame Test Adapter (No Cisco)

This page is meant to be loaded inside ServiceNow OpenFrame. It initializes OpenFrame, creates an Incident via Table API, and screen-pops the created record.

REST API Explorer uses your ServiceNow session, but this external page typically needs credentials to call Table API. Ensure your user has permission to create incidents.
